# HF Hub Scout

A faster, cleaner way to discover Hugging Face models. 
I built this because the Hub has800k+ models and it needed more defined filters to enable users to find the model which is right for them.

**What it does**
- Queries the public HF Hub API in real-time
- Sorts by downloads, likes, trending, or newest
- Filters by ML task (NLP, vision, speech, etc.)
- Works on mobile

**Why I built it**

I wanted to prove I could ship a full product loop(API → UI → deploy) using the HF ecosystem.

**Stack:** Vanilla JS, CSS Grid, HF Hub API.
